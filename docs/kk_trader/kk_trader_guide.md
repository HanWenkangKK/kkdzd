# 使用教程

本教程分为前期准备、软件使用说明两部分，可从右侧目录跳转

## 前期准备

### 系统分辨率设置
脚本仅支持**16:9**或**16:10**分辨率上运行。

请查看系统分辨率是否符合上述要求，在桌面上点击**右键**，选择**显示设置**

<img src="/images/显示设置2.png" alt="Pulpit rock" width="300" height="228">

查看显示器分辨率

<img src="/images/查看显示器分辨率.png" alt="Pulpit rock" width="600" height="228">

比如此处为2560x1440，符合16:9的要求，不需要修改。

假设此处为1920x1440，则不符合要求，需要您更改为1920x1080或者1920x1200以符合要求。

### 买入条件设置


::: info 价格设置
这里设置你预期的买入价格范围。
<img src="/images/选项设置2.png" alt="Pulpit rock" width="600" height="228">

::: details 使用技巧
例如：当前你想购买的物品的价格为189000，你想等它价格变成187000以下时进行抢购，这里可以设置为150000~187000。

这里设置一个150000作为价格下限是必要的。由于软件的数字识别有很低的概率把价格识别缺一位，如把170000识别成70000，这时设置价格下限可能防止错误的购买。所以强烈建议价格下限不要设置为0。
:::


::: info 同时检测第一根柱子数量
<img src="/images/选项设置3.png" alt="Pulpit rock" width="600" height="228">
开启后，软件会检测第一个柱子的数量值，如下图，当数量符合你设置的范围时才会购买，不符合则不会购买。
<img src="/images/选项设置4.png" alt="Pulpit rock" width="300" height="228">
此选项开启后会显著影响购买时的速度，待优化，不需要时请不要勾选
:::

::: info 拉高购买数量
<img src="/images/选项设置5.png" alt="Pulpit rock" width="600" height="228">
开启后，软件每次刷新页面都会自动拉高下图中的数量进度条到你设置的百分比，以实现单次购买多个商品。假如你想每次抢购200发子弹，或者5个配件，请勾选它，并设置百分比为100（拉满）。
<img src="/images/选项设置6.png" alt="Pulpit rock" width="500" height="228">
:::

### 自动停止设置
此处提供三种自动停止抢购的方案，每项具体功能已经在软件上写明，可以按需开启
<img src="/images/自动停止设置1.png" alt="Pulpit rock" width="700" height="228">

### 刷新设置
由于每个人的电脑配置、网速不同，每次刷新进入购买页面的速度也不同，这里设置每次刷新页面后等待的时间。

::: info 稳定模式
<img src="/images/刷新设置1.png" alt="Pulpit rock" width="600" height="228">
开启后软件会自动以合适的刷新速度进行刷新。如果关闭稳定模式，请在自行测试后在下方设置合适的刷新延迟，以保证软件运行时不会出错。
:::

::: info 启用模式切换
<img src="/images/刷新设置2.png" alt="Pulpit rock" width="600" height="228">
由于三角洲交易行会越刷越卡，开启模式切换后，会每隔一段时间自动切换一次大战场模式，可以一定程度上减少这种越刷越卡的情况
:::
