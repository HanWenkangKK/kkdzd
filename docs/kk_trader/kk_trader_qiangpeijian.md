# 关于抢配件

鉴于近期各位玩家抢配件养号的需求，特地说明一下抢配件时软件的设置方案。阅读本文之前请先看完“使用教程“文档

假如你要抢补货价的共鸣消音，它的补货价抢到手只需要50000多，但补货时刷到的价格并非补货价，而是仅仅比当前价格稍低一些的价格。
例如当前共鸣消音价格为187000，在补货时会出现新的最低价的柱子，可能会显示一个172310这样的价格，而非五万的价格。
那么我们只需要在脚本的价格设置里填上130000~176000这样一个区间，那么当补货时，软件就会帮我们抢购，买到手的就会是五万多的补货价。

<img src="/images/抢配件结果.png" alt="Pulpit rock" width="400" height="328">

如何正确地设置这个价格区间？我们正式抢购之前，可以先设置一个较低的价格区间，如0~100，这样启动脚本，脚本会只刷新并检测价格，但并不抢购。这样刷新一段时间后，翻看脚本的日志，会显示这段时间内刷到的最低价是多少，假设最低价是167320，那么补货价就在这个价格附近，我们就可以依据这个价格来设置抢购价格范围了，比如设置为150000~169000。
