# 常见问题 FAQ

## 1、按F12启动后没效果/点击一下就不动了
大概率是没有使用管理员身份启动软件

## 2、刷新时频繁退出交易行
推荐使用稳定模式。可能是刷新久了游戏卡顿，重启游戏即可解决。请善用软件的**模式切换**功能以防止交易行变卡

## 3、软件运行一段时间后变慢变卡
这是因为三角洲交易行有内存溢出的bug，运行一段时间后游戏会变卡，如果抢购一段时间后卡顿，建议重启游戏再试。请善用软件的**模式切换**功能以防止交易行变卡

## 4、开启柱子检测后购买速度变慢
目前检测柱子这个选项的速度有待优化，如果可以尽量不要使用
